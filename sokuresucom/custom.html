1.APIキーの取得


1.1 ログイン


以下のURLからchatgptアカウントに新規登録またはログインしてください。
GPT - OpenAI API https://platform.openai.com/overview

1.2 GPTのAPIキーを発行する。


ログインしたら右上の"Personal"から"View API keys"へ移動し "Create New Secret Key"をクリックします。 任意の文字を入れたらAPIキーが発行されます。
※注意！ APIキーはこの時しか表示されません。忘れずにメモしておきましょう！
※注意！ APIキーは個人ごとに別々に発行されています。誰にも知られないようにしましょう！

1.3 クレジットカードの登録


同じ画面から"Billing"、"Payment methods"から"Add payment method"を選択し、クレジットカード番号を入力しましょう。 ※注意！ クレジットカード登録時点で最低5ドルの支払いが必要です。(2023/10/8 時点) （任意）利用額上限の設定 同じ画面の"Usage limits"から毎月の利用額の上限が決められます。
Soft limitは指定した額を超えるとメールアドレスに通知が来るようになり、 Hard limitを指定するとそれを超える利用はできなくなります。

2.利用


2.1 APIキーの設定


1.2で発行したAPIキーをコピーして、「APIキーを入力」部分に貼り付けましょう。

2.2 入力文と言い方設定


「APIキーを入力」の下に返答を考えたい文章を入れてください。 また言い方設定から敬語かタメ口かを設定できます。

2.3 実行


「送信」ボタンを押します。下に返答の3つの候補が出ます。
